# BCCL DRMS Project

## Digital Record Management System

This project is developed for Bharat Coking Coal Limited (BCCL) to manage office documents and records digitally.

## Features
- User Login Authentication
- Dashboard
- Add New Record
- View Records
- Edit Records
- Generate Reports
- SQLite Database Storage

## Technologies Used
- Python
- Flask
- SQLite
- HTML
- CSS

## Project Structure
- app.py : Main application file
- auth.py : Handles login authentication
- db.py : Database connection and operations
- database.db : SQLite database file
- templates/ : HTML templates
- static/ : CSS files

## How to Run the Project

1. Install Python
2. Install Flask using:
   pip install flask
3. Run the application:
   python app.py
4. Open browser and go to:
   http://127.0.0.1:5000/

## Project Objective
The objective of this project is to replace manual document handling with a digital record management system.
